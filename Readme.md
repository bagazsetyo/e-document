# Instalasi 

## 1. Alat dan Bahan
1. Git
2. Laragon versi terbaru

## 2. Clone Repository
```bash
git clone
```

## 3. Menjalankan Aplikasi
1. Buka Laragon
2. Klik start all

## 4. Membuat Database
1. Buka Laragon
2. Klik Database
3. Buat database baru dengan nama `edocument`
4. Import file `edocument.sql` yang ada di folder `database`

## 5. Menjalankan Aplikasi
1. Buka browser
2. Ketikkan `http://edocument.test/`
